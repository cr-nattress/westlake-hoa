# Story 10.5: Records Custodian Section

## User Story

**As a** homeowner seeking HOA records
**I want** to know what records Bold maintains and how to access them
**So that** I can exercise my rights to inspect HOA records under CCIOA

## Description

Create a component that displays the types of records Bold Property Management maintains as custodian for the HOA, along with guidance on the access process. This empowers homeowners to make informed records requests.

## Acceptance Criteria

- [ ] Component displays list of record types Bold maintains
- [ ] Brief description for each record type
- [ ] Access process information (statutory timelines, business hours)
- [ ] Link to Records Request Guide page
- [ ] Note about records Bold does NOT maintain (privileged documents)
- [ ] Clean, informative layout
- [ ] Accessible

```tsx
export function RecordsCustody({ records, accessRequirements, className }: RecordsCustodyProps) {
  return (

          Records Maintained by Bold

        <CardDescription>

        </CardDescription>
      </CardHeader>
      <CardContent className="space-y-6">
        <div className="grid gap-3">
          {records.map((record, index) => (

              <Folder className="h-4 w-4 text-muted-foreground mt-1 shrink-0" />

            </div>

      </CardContent>
    </Card>
  );
```
