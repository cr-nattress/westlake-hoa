# Story 10.4: Specific Contact Details Enhancement

## User Story

**As a** homeowner needing to contact Bold Property Management
**I want** specific email addresses with guidance on which to use
**So that** my communication reaches the right destination and gets a response

## Description

Create a component that displays Bold's specific email addresses with clear guidance on what each is used for. This goes beyond the generic email domains currently shown and helps homeowners communicate effectively.

## Acceptance Criteria

- [ ] Component displays specific email addresses (not just domains)
- [ ] Each email has clear purpose/usage guidance
- [ ] Notes about what kind of communications use each address
- [ ] Copy-to-clipboard functionality for emails
- [ ] Visual distinction between email types
- [ ] Mobile-friendly layout
- [ ] Integrates with existing contact display

### Component Structure

```tsx
export function ManagementContacts({ contacts, showMailingAddress, className }: ManagementContactsProps) {
  const copyEmail = (email: string) => {
    navigator.clipboard.writeText(email);
    toast.success("Email copied to clipboard");
  };

  return (
    <Card className={className}>
      <CardHeader>
        <CardTitle className="flex items-center gap-2">
          <Mail className="h-5 w-5" />
          Contact Bold Property Management
        </CardTitle>
      </CardHeader>
      <CardContent className="space-y-4">
        {contacts.map((contact, index) => (
          <div key={index} className="space-y-1">
            <div className="flex items-center justify-between">
              <span className="font-medium">{contact.purpose}</span>
              <Button
                variant="ghost"
                size="sm"
                onClick={() => copyEmail(contact.email)}
              >
                <Copy className="h-4 w-4" />
              </Button>
            </div>
            <a
              href={`mailto:${contact.email}`}
              className="text-primary hover:underline text-sm break-all"
            >
              {contact.email}
            </a>
            {contact.notes && (
              <p className="text-sm text-muted-foreground">{contact.notes}</p>
            )}
          </div>
        ))}

        {showMailingAddress && (
          <div className="pt-4 border-t">
            <h4 className="font-medium mb-2">Mailing Address</h4>
            <AddressDisplay address={BOLD_MAILING_ADDRESS} />
          </div>
        )}
      </CardContent>
    </Card>
  );
}
```
