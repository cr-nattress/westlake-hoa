# Story 12.5: Add "When to Contact" Guide

## User Story

**As a** homeowner unsure who to contact
**I want** a guide explaining when to contact each entity
**So that** I reach the right person the first time

## Description

Add a collapsible "When to Contact" section that helps homeowners understand which entity handles what type of issue. This reduces misdirected inquiries and sets proper expectations.

## Acceptance Criteria

- [ ] "When to Contact" section above contact cards
- [ ] Collapsible/expandable accordion format
- [ ] Clear examples of issues for each entity
- [ ] Visual indicators (icons) for quick scanning
- [ ] Responsive layout
- [ ] Defaults to expanded on desktop, collapsed on mobile

## Content

### Property Management (Bold Solutions)
**Contact for:**
- General property questions
- Maintenance requests and issues
- Pest control scheduling
- Access requests to units
- Payment questions (non-legal)
- Community rule questions
- Vendor coordination
- HOA notices and communications

### Insurance (Mountain West)
**Contact for:**
- Filing insurance claims
- Certificate of insurance requests
- Coverage questions
- Loss reports
- Insurance policy questions

### Legal Counsel (Alpenglow Law)
**Contact for:**
- Collections matters (when referred)
- Legal disputes
- Governance compliance questions
- Formal legal correspondence

**Note:** Most homeowners should NOT contact legal counsel directly. Matters are typically escalated by the Board or management.

### HOA Board
**Contact for:**
- Policy questions or concerns
- Appeals of decisions
- Governance matters
- Matters not resolved by management
- Annual meeting topics

```typescript
// src/components/when-to-contact.tsx
import {
  Accordion,
  AccordionContent,
  AccordionItem,
  AccordionTrigger,
} from "@/components/ui/accordion";
import { Building2, Shield, Scale, Users } from "lucide-react";

const CONTACT_GUIDE = [
  {
    id: "management",
    icon: Building2,
    title: "Property Management",
    entity: "Bold Solutions",
    items: [
      "General property questions",
      "Maintenance requests and issues",
      "Pest control scheduling",
      "Access requests to units",
      "Payment questions (non-legal)",
      "Community rule questions",
      "Vendor coordination",
      "HOA notices and communications",
    ],
  },
  {
    id: "insurance",
    icon: Shield,
    title: "Insurance",
    entity: "Mountain West Insurance",
    items: [
      "Filing insurance claims",
      "Certificate of insurance requests",
      "Coverage questions",
      "Loss reports",
      "Insurance policy questions",
    ],
  },
  {
    id: "legal",
    icon: Scale,
    title: "Legal Counsel",
    entity: "Alpenglow Law",
    items: [
      "Collections matters (when referred)",
      "Legal disputes",
      "Governance compliance questions",
      "Formal legal correspondence",
    ],
    warning: "Most homeowners should NOT contact legal counsel directly. Matters are typically escalated by the Board or management.",
  },
  {
    id: "board",
    icon: Users,
    title: "HOA Board",
    entity: "Board of Directors",
    items: [
      "Policy questions or concerns",
      "Appeals of decisions",
      "Governance matters",
      "Matters not resolved by management",
      "Annual meeting topics",
    ],
  },
];

export function WhenToContact() {
  return (
    <section className="mb-8">
      <h2 className="text-lg font-semibold mb-4">When to Contact</h2>
      <Accordion type="multiple" defaultValue={["management"]} className="w-full">
        {CONTACT_GUIDE.map((section) => (
          <AccordionItem key={section.id} value={section.id}>
            <AccordionTrigger className="hover:no-underline">
              <div className="flex items-center gap-3">
                <section.icon className="h-5 w-5" />
                <span>{section.title}</span>
                <span className="text-muted-foreground text-sm font-normal">
                  ({section.entity})
                </span>
              </div>
            </AccordionTrigger>
            <AccordionContent>
              <ul className="list-disc list-inside space-y-1 text-sm ml-8">
                {section.items.map((item, i) => (
                  <li key={i}>{item}</li>
                ))}
              </ul>
              {section.warning && (
                <p className="text-sm text-amber-600 dark:text-amber-400 mt-3 ml-8">
                  {section.warning}
                </p>
              )}
            </AccordionContent>
          </AccordionItem>
        ))}
      </Accordion>
    </section>
  );
}
```
