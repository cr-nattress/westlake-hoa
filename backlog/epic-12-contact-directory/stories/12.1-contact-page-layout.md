# Story 12.1: Create Contact Page Layout & Route

## User Story

**As a** homeowner looking for contact information
**I want** a dedicated contacts page
**So that** I can quickly find the right person or entity to contact

## Description

Create the `/contacts` route with the main page layout, including all four contact categories (Property Management, Insurance, Legal, Board) and proper organization with visual hierarchy.

## Acceptance Criteria

- [ ] New page at `/contacts` route
- [ ] Page title and description meta tags set
- [ ] Four main sections: Management, Insurance, Legal, Board
- [ ] Each section clearly labeled with category header
- [ ] Responsive grid layout for contact cards
- [ ] Proper spacing and visual hierarchy
- [ ] Disclaimer section at bottom
- [ ] "Last updated" date displayed
- [ ] Dark mode compatible

## Technical Details

### Page Component

```typescript
// src/app/contacts/page.tsx
import { Metadata } from "next";
import { ContactSection } from "@/components/contact-section";
import {
  PROPERTY_MANAGEMENT_FULL,
  INSURANCE_BROKER_FULL,
  LEGAL_COUNSEL_FULL,
  HOA_BOARD_FULL,
  KNOWLEDGE_BASE_METADATA
} from "@/lib/data/contact-directory";

export const metadata: Metadata = {
  title: "Contact Directory | Westlake Village HOA",
  description: "Find contact information for property management, insurance, legal counsel, and HOA board.",
};

export default function ContactsPage() {
  return (
    <div className="container mx-auto px-4 py-8">
      <header className="mb-8">
        <h1 className="text-3xl font-bold mb-2">Contact Directory</h1>
        <p className="text-muted-foreground">
          Find the right contact for your HOA needs
        </p>
      </header>

      <div className="space-y-8">
        <ContactSection
          title="Property Management"
          contact={PROPERTY_MANAGEMENT_FULL}
        />
        <ContactSection
          title="Insurance"
          contact={INSURANCE_BROKER_FULL}
        />
        <ContactSection
          title="Legal Counsel"
          contact={LEGAL_COUNSEL_FULL}
        />
        <ContactSection
          title="HOA Board"
          contact={HOA_BOARD_FULL}
        />
      </div>

      <footer className="mt-12 pt-6 border-t">
        <p className="text-sm text-muted-foreground">
          This is an unofficial website. Contact information compiled from
          official HOA documents for homeowner convenience.
        </p>
        <p className="text-xs text-muted-foreground mt-2">
          Last updated: {KNOWLEDGE_BASE_METADATA.lastUpdated}
        </p>
      </footer>
    </div>
  );
}
```
