# Story 9.3: Governance & Board Information Section

## User Story

**As a** homeowner wanting to understand HOA governance
**I want** to see who leads the HOA and how decisions are made
**So that** I understand the authority structure and can engage appropriately

## Description

Create a governance section that displays board composition, board authority, and decision-making processes. This can be a dedicated page or a major section on the existing About page.

## Acceptance Criteria

- [ ] Board member(s) listed with position and "as of" date
- [ ] Board authority/powers clearly listed
- [ ] Decision-making process explained (meetings vs. written consent)
- [ ] Required vote recordings noted (collections, legal, foreclosure)
- [ ] Clear, neutral tone maintained
- [ ] "As of" dates prominently displayed for time-sensitive info
- [ ] Integrated with /about page or new /governance page

```typescript
// In src/lib/data/institutional-knowledge.ts

export interface BoardMember {
  name: string;
  position: string;
  asOfDate: string;  // e.g., "November 2025"
}

export const BOARD_MEMBERS: BoardMember[] = [
  {
    name: "Mark Matz",
    position: "President",
    asOfDate: "November 2025"
  }
  // Other positions may be added when known
];

export const BOARD_AUTHORITY = {
  powers: [
    "Adopts and amends policies, procedures, and rules",
    "Authorizes collections, legal actions, and referrals",
    "Retains legal counsel",
    "Oversees property management"
  ],
  decisionMaking: {
    methods: [
      "Board meetings (regular or special)",
      "Unanimous written consent"
    ],
    recordedVotes: [
      "Collections referrals to legal counsel",
      "Authorization of legal actions",
      "Foreclosure authorization"
    ]
  }
};
```

### Component

```typescript
// src/components/board-info.tsx
export function BoardInfo() {
  return (
    <section>
      <h2>Board of Directors</h2>

      <div className="board-members">
        {BOARD_MEMBERS.map(member => (
          <BoardMemberCard key={member.name} member={member} />
        ))}
      </div>

      <div className="board-authority">
        <h3>Board Authority</h3>
        <ul>
          {BOARD_AUTHORITY.powers.map(power => (
            <li key={power}>{power}</li>
          ))}
        </ul>
      </div>

      <div className="decision-making">
        <h3>How Decisions Are Made</h3>
        <p>The Board may act through:</p>
        <ul>
          {BOARD_AUTHORITY.decisionMaking.methods.map(method => (
            <li key={method}>{method}</li>
          ))}
        </ul>

        <h4>Recorded Votes Required For:</h4>
        <ul>
          {BOARD_AUTHORITY.decisionMaking.recordedVotes.map(item => (
            <li key={item}>{item}</li>
          ))}
        </ul>
      </div>
    </section>
  );
}
```
