# Story 9.2: Key Contacts Page

## User Story

**As a** homeowner needing to contact the HOA
**I want** a single page with all key contact information
**So that** I can quickly find who to contact for different issues

## Description

Create a dedicated `/contacts` page that consolidates all HOA-related contact information including property management, legal counsel, insurance broker, and HOA mailing addresses. This becomes the central hub for "who do I contact" questions.

## Acceptance Criteria

- [ ] New page at `/contacts` route
- [ ] Page displays all contact entities from knowledge base
- [ ] Each contact shows: name, role, address, email domains (if any), phone (if any)
- [ ] Contacts organized by type (Management, Legal, Insurance, HOA)
- [ ] Clear visual hierarchy distinguishing different contacts
- [ ] Mobile-responsive card layout
- [ ] Added to main navigation
- [ ] Page includes disclaimer about unofficial nature
- [ ] "Last updated" date visible
