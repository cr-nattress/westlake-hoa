# Story 9.1: HOA Identity & Legal Structure Section

## User Story

**As a** homeowner or prospective buyer
**I want** to see the HOA's official legal name and structure
**So that** I understand the entity I'm dealing with and can reference it correctly in formal communications

## Description

Add a prominent section displaying the HOA's legal identity, including its full legal name, DBA, jurisdiction, entity type, and basic property statistics. This information should be visible on the About page and potentially on the homepage.

## Acceptance Criteria

- [ ] HOA legal name displayed: "Benchmark Condominium Homeowners Association"
- [ ] DBA displayed: "Westlake Village Condominium Association, Inc."
- [ ] Jurisdiction listed: CCIOA and Colorado Nonprofit Corporation Act
- [ ] Entity type shown: Condominium Association (Non-profit corporation)
- [ ] Property stats: ~92 units, Westlake Village Condominiums
- [ ] Section has appropriate heading and styling
- [ ] Information sourced from data file (not hardcoded in component)
- [ ] Accessible and responsive design
