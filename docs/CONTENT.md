## What Owners Must Insure

The HOA's master policy does **NOT** cover everything. Individual unit owners are responsible for insuring:

| Owner Responsibility | Description |
|---------------------|-------------|
| Interior Improvements | Any upgrades beyond original build-out |
| Personal Property | Furniture, electronics, belongings |
| Loss of Use | Alternative housing if unit is uninhabitable |
| Loss of Rental Income | If you rent your unit |
| Loss Assessments | Special assessments from HOA claims |
| Personal Liability | Consider umbrella policy extension |

### What the Association Insures

- Common elements (hallways, exterior, roof, etc.)
- Limited common elements (decks, patios assigned to units)
- Original unit build-out per declarations
